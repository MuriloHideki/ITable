# ITable
Este repositório contém o código backend do sistema de reservas de mesas para restaurantes, desenvolvido em Node.js. O sistema gerencia reservas, horários e mesas disponíveis para facilitar o fluxo operacional do restaurante.

Clone o repositório:

```bash
# Clonar o repositório
git clone <URL_DO_REPOSITORIO>
cd backend-reservas

# Instalar dependências
npm install

# Iniciar o servidor local
npm start
